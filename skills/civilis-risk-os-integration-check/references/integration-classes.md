# Integration Classes

## Live Challengeable Settlement

Use this label only if:

- buyer / seller / evaluator are explicit
- principal routes through protected settlement
- claim and resolve both exist
- later repricing is part of the flow

## Quote-Only Consumer

Use this label if:

- the consumer only needs quote intelligence or recommendation
- no protected settlement or dispute flow is implemented

## Portability Adapter

Use this label if:

- the consumer normalizes another commerce surface into the Risk OS schema
- but does not yet prove a live protected settlement loop

## UNVERIFIED

Use this label if:

- key proof or settlement requirements are unknown
- claims are broader than evidence
